# analyze-ai-agent-codebase

Analyze open-source AI agent repositories by focusing on:

- architecture layers and dependency direction
- core type contracts and registries
- loop and tool execution paths
- extension surfaces and safety boundaries
- design trade-offs and modification paths

## Files

- `SKILL.md`: main workflow and triggering description
- `agents/openai.yaml`: Codex-compatible skill metadata
- `agents/agent.yaml`: mirror metadata for marketplace-style repo structure
- `references/pi-methodology.md`: methodology extracted from the `pi` architecture book
- `references/analysis-template.md`: reusable report template
